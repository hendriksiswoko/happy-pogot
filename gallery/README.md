Folder ini khusus untuk menyimpan foto galeri landing page.

Cara menambah foto:
1. Masukkan file gambar ke folder ini.
2. Gunakan nama file seperti 1.jpg, 2.jpg, 3.jpg, dan seterusnya.
3. Tambahkan atau ubah elemen img di index.html sesuai nama file yang ingin ditampilkan.

Rekomendasi:
- Gunakan gambar dengan rasio 1:1 untuk hasil terbaik.
- Format yang disarankan: JPG atau WebP.
- Ukuran ideal: 800x800 px atau lebih.
